Environment setup:
    conda create -n llaga python=3.10
    conda activate llaga
    cd /scratch/jl11523/projects/LLaGA
    ./install.sh


Generate prompt jsonl files:
    projects/LLaGA/dataloader/data_preprocess.py


Train on specific dataset:
    projects/LLaGA/scripts/train_{datasetname}.SBATCH
    (--pretrained_embedding_type {consistent with ../dataset/name/{pretrained_embedding_type}_feat.pt})


Evaluate:
    projects/LLaGA/scripts/eval_{datasetname}.SBATCH


Calculate metrics:
    projects/LLaGA/scripts/cal_metric.SBATCH
    ( --emb_type {textimage|image|text} )
    
    cal_metric_cora时每次新的eval文件更新在这个参数下： --res_path <new_eval_result_file>
